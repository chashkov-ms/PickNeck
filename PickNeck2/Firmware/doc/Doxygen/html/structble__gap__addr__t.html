◆ addr_id_peer

uint8_t ble_gap_addr_t::addr_id_peer

Only valid for peer addresses. This bit is set by the SoftDevice to indicate whether the address has been resolved from a Resolvable Private Address (when the peer is using privacy). If set to 1, addr and addr_type refer to the identity address of the resolved address.

This bit is ignored when a variable of type ble_gap_addr_t is used as input to API functions.
